<h1>QR Code Component</h1>
<h2>Description</h2>
This project features a responsive QR Code component that directs users to the Frontend Mentor website. The component showcases a modern design, suitable for various screen sizes, ensuring a seamless user experience.

<h2>Features</h2>
<ul>
<li><h3>Responsive Design:</h3> Adjusts to different screen sizes using CSS Flexbox and Media Queries.</li>
<li><h3>QR Code Integration:</h3> Utilizes a QR code image that links directly to the Frontend Mentor website.</li>
<li><h3>Styled Component:</h3> Features a clean and minimalist aesthetic with hover effects for improved user interaction.</li>
<li><h3>Accessibility:</h3> Includes alt text for the QR code to enhance accessibility.</li>
  </ul>
Technologies Used
<ul>
  <li>HTML</li>
  <li>CSS</li>
</ul>
